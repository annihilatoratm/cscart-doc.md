# [Payment Flow](https://github.com/annihilatoratm/opencart-doc/blob/main/doc-eng.md#payment-flow-1)

  ## Payment Flow

  1. Please select the needed product from the main menu.

  <img src="/images/cscart-1.png" width=60% height=60%>
  
  2. Once the item was selected, click on "Add to card" button to add the product to the shop cart. Pop-up message will provide with two options: select *Continue shopping* to continue explore the shop and *Checkout* to start the payment flow. 

  <img src="/images/cscart-2.png" width=60% height=60%>
  <img src="/images/cscart-3.png" width=60% height=60%>
  
  3. By clicking on the *My cart* button above, you can either explore the cart in *View cart* option or start the payment flow by clicking on *Checkout* button.

  <img src="/images/cscart-4.png" width=60% height=60%>

  4. If chosen, the **Card contents** section looks like this:

  <img src="/images/cscart-5.png" width=60% height=60%>

  5. Once *Checkout* was chosen, please fill all required parameters and press on *Place order* button.

  <img src="/images/cscart-6.png" width=60% height=60%>
  
  6. Fill all required parameters on the form and press the Process payment button to finish the payment process.

  <img src="/images/cscart-7.png" width=60% height=60%>
  <img src="/images/cscart-8.png" width=60% height=60%>
